# Roadmap

wAItwise is still a prototype. The next improvements are mostly about retrieval quality, review depth, and product polish.

## Near Term

- Improve retrieval relevance across broader conversations.
- Expand and clean the topic-index corpus.
- Add stronger manual verification around ChatGPT generation detection.
- Make review history more useful for spaced repetition.
- Add clearer provider error states in the popup.

## Product Questions

- How often should a quiz appear during a long ChatGPT session?
- When should wAItwise avoid interrupting the user?
- Should the pet system reward attempts, accuracy, streaks, or review consistency most heavily?
- Should Retrieval Review stay focused on ML/CV interview material or become user-configurable?

## Engineering Questions

- Should provider prompts become versioned fixtures?
- Should corpus metadata be generated from a repeatable import script?
- Should detector behavior get browser automation smoke tests?
- Should quiz history support export or sync later?
